# Data-Science-in-Manufacturing

### **內容概述**

此網頁為製造資料科學的R程式實作教學，深入探討不同類型的製造資料，結合製造、統計、機器學習以及領域知識，做出最精闢的資料分析．

指導老師為[李家岩](http://polab.imis.ncku.edu.tw/Bio.html)老師
